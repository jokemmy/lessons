## 1.3 表达式和运算符

---

### 原始表达式

最简单的表达式，包含常量、直接量、关键字和变量。

```js
"hello"         // 返回一个字符串
true            // 返回布尔值
null            // 返回null
i               // 返回变量 i 的值
undefined       // 返回 undefined
```
---

### 对象和数组初始化表达式

数组和对象的初始化表达式实际上是一个新创建的数组和对象

```js
[]                // 创建了一个空数组
[ 1, 2, 3 ]       // 创建了一个有三个元素的数组
{}                // 创建了一个对象
{ x: 1, y: 2 }    // 创建了一个有两个属性的对象
```

---

### 定义函数表达式

```js
var addOne = function( x ) { return x + 1; };
```
---

### 访问属性表达式

```js
var o = { x: 1, y: 2 };
o.x                         // 返回 o 的 x 属性
o['y']                      // 返回 o 的 y 属性
o.z                         // 返回 undefined
```

---

### 调用表达式

执行函数或方法的表达式

```js
f( 0 )          // 执行了 f 函数
Math.abs( -2 )  // 取绝对值 2
Date.now()      // 获取当前时间戳
```

---

### 创建对象

```js
new Date()        // 创建一个日期对象
new Object()      // 创建一个空对象
new Object        // 创建时不需要参数时括号可以省略
```

---

### 左值

### 运算符结合性
